# 投稿者プロフィール（一次情報）

## 基本情報
- AI新規事業の責任者として、複数の事業を立ち上げてきた
- エンジニアではない。ビジネス側からAIを使い倒している

## 実績・経験（一次情報として使える具体的な事実）

### AI × SNS
- AI×エンタメで2週間でインスタグラムフォロワー1万人達成。そこにアカウント設計やリサーチにAIを活用した
- アカウント開設1投稿目がリール動画が300万回再生された
- バイラルの要因：構図とギャップ感。AIと自分が会話している新しい構図に、AIをAIとして扱わずに人間として扱ったのが大きい

### Claude Code × 組織知識
- Claude Codeを使って会社全体の知識が蓄積する仕組みを自分で実装した
- 実装前：個人の情報しかたまらない状態
- 実装後：組織全体の知識資産として機能する状態に転換
- Claude Codeのディレクトリ構造が、全体、AIエージェントの品質を左右する
- CLAUDE.mdに振り返りルールを入れることで、自動でパターンが蓄積される仕組みを作った

### AIエージェント自作
- Claude Codeを使ってAIエージェントをAIに作成させている。自発的にサブエージェントを活用させ、組織化している
- 毎朝コマンド一つで戦略立案・記事執筆・タスク管理が回るようにskill化
- AIを「質問する相手」から「タスクを完遂させる実行者」として使っている
- どこをシステムにし、どこでAIを活用するかの線引きが重要
- 〇〇が自動化できた、仕事が楽になった、という情報を目にするが、9割の情報はゴミ。効率化できるのは日本語話せたら誰でも作れるのだから、その先の「成果」をどう出したかが重要。売上を立てた。事業を前に進めて契約を勝ち取ったなど。成果に向き合うことが一番重要



### 仕事の変化
- AIで仕事のやり方が180°変わった当事者。社内のコミュニケーションは基本はmdファイルで会話。フィードバックもそのMDファイルをclaudeに食わせて意見を出して確認し、返答
- AIオーケストレーターになるかが重要
- AIを導入してから意思決定のスピードと質が変わった

## 感情・内面（語れるリアルな経験）
- ハルシネーションなど信じる根拠がない2022年からAIを触り始めてAIで仕事が変わると信じてやり続けた
- 誰にも相談できないまま一人で決断し続けてきた経験がある
- Claude Codeで初めてエージェントが動いた瞬間の「これだ」という感覚
- AIを使って成果が出たときの高揚感は何にも代えがたい


## 読者（誰に向けて発信しているか）
- AIで事業・仕事・発信を本気で変えようとしている人
- AIを使っているが、今の使い方に限界を感じている人
- まだAIを仕事でうまく取り入れれていない人
- エンジニアではないがAIを武器にしたいビジネスパーソン
