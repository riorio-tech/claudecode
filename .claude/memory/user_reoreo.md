## 役割・目標
- TikTok Shop の GMV 月間 1,000万円 達成を目指す事業者
- 商品動画の自動生成パイプラインを構築・運用している
- AI ツール（Claude Code / fal.ai / ElevenLabs）を実務で使いこなすレベル

## 作業スタイル
- 日本語でやり取りする
- 返答は短く・直接的に。前置き・まとめ不要
- コードを見せるより動かして確認したい（「作成して」で実行まで求める）
- 問題が起きたら自律的に原因を特定・修正することを期待している
- 「改善して」と言ったら、コード変更 + 再実行まで含む

## 技術レベル

- API 統合（fal.ai / Anthropic / ElevenLabs）の経験あり
- git は使うが、コミットは「コミットして」と明示したときのみ
